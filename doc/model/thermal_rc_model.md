# HotSpot热RC模型文档

## 1. 引言

HotSpot是一款 pre-RTL 热仿真器，用于 VLSI 设计中的热分析。其核心思想是将热传导问题转化为电路问题，通过 RC 网络进行建模和求解。本文档详细介绍 HotSpot 所使用的 4 层垂直热 RC 模型的物理背景、数学推导以及数值方法。

---

## 2. 热传导物理背景

### 2.1 Fourier 定律

热传导的基本定律是 Fourier 定律，描述了热通量与温度梯度之间的关系：

```
q = -k · ∇T
```

其中：
- **q** 是热通量，单位 W/m²
- **k** 是热导率，单位 W/(m·K)
- **T** 是温度，单位 K

负号表示热量从高温区向低温区传递。

### 2.2 能量守恒方程

在三维空间中，考虑介质的热容效应，能量守恒方程为：

```
ρ · cₚ · ∂T/∂t = ∇ · (k · ∇T) + Q
```

其中：
- **ρ** 是材料密度，单位 kg/m³
- **cₚ** 是比热容，单位 J/(kg·K)
- **Q** 是体积热源，单位 W/m³
- **t** 是时间，单位 s

该方程表明：单位体积内热能的积累率 = 热传导的净流入率 + 内部热源的产热率。

### 2.3 边界条件

在实际问题中，需要设定边界条件：

- ** Dirichlet 边界条件**：设定边界温度 T = T₀
- ** Neumann 边界条件**：设定边界热通量 q = q₀
- ** 绝热边界条件**：q = 0（热通量为零）

HotSpot 中默认设定环境温度作为 Dirichlet 边界条件。

## 3. RC 网络抽象

### 3.1 热-电类比

热传导问题与电路问题具有天然的类比关系。通过这种类比，可以将热问题转化为熟悉的电路问题来求解。

| 热学量 | 电学类比 | 符号 | 单位 |
|--------|----------|------|------|
| 温度 T | 电压 V | T | K |
| 热通量 q | 电流密度 J | q | W/m² |
| 热导率 k | 电导率 σ | k | W/(m·K) |
| 热阻 Rₜₕ | 电阻 R | R | K/W |
| 热容 Cₜₕ | 电容 C | C | J/K |
| 热流量 Φ | 电流 I | Φ | W |

### 3.2 基本 RC 单元

一个基本的热 RC 单元由一个电阻 R 和一个电容 C 组成：

```
    I ----→|----R----+----←----
           |          |
           C          V
           |          |
           +----→-----+
```

- 电阻 R 两端的电压降与通过的热流量成正比：ΔV = R · Φ
- 电容 C 存储的热能为：E = C · T

### 3.3 热阻的定义

热阻是描述材料阻止热量传递能力的物理量。对于均匀材料：

```
R = ΔT / Φ
```

其中 ΔT 是温降，单位 K；Φ 是热流量，单位 W。

### 3.4 热容的定义

热容是描述材料存储热能能力的物理量：

```
C = Q / ΔT
```

其中 Q 是吸收的热量，单位 J；ΔT 是温度变化，单位 K。

## 4. HotSpot 的 4 层模型

### 4.1 模型结构

HotSpot 采用 4 层垂直堆叠模型，从芯片（silicon）到散热器（heatsink）：

```
┌─────────────────────────┐
│      HSINK (层 3)        │  散热器层
│   材料: 铜 (copper)       │
│   厚度: 6.9 mm           │
├─────────────────────────┤
│       HSP (层 2)         │  扩散器层
│   材料: 铜 (copper)       │
│   厚度: 1.0 mm           │
├─────────────────────────┤
│      IFACE (层 1)        │  界面层
│   材料: TIM (热界面材料)   │
│   厚度: 20 μm            │
├─────────────────────────┤
│     SILICON (层 0)       │  芯片层
│   材料: 硅 (silicon)      │
│   厚度: 0.15 mm          │
│   ← 热源施加处            │
└─────────────────────────┘
         ↓ 热流方向
```

### 4.2 各层功能

**芯片层 (SILICON)**
- 承载电路元件，是主要的热源产生处
- 厚度仅 0.15 mm，热量密度最高
- 硅的热导率约为 150 W/(m·K)

**界面层 (IFACE)**
- 填充芯片与扩散器之间的空隙
- 厚度很薄（20 μm），但热阻通常较大
- 使用的 TIM 材料热导率约为 1-10 W/(m·K)

**扩散器层 (HSP)**
- 将热量从集中区域扩散到更大面积
- 厚度 1 mm，材料为铜
- 铜的热导率约为 400 W/(m·K)

**散热器层 (HSINK)**
- 最终将热量传递到环境空气
- 厚度 6.9 mm，材料为铜
- 通常配有鳍片结构以增大散热面积

### 4.3 热流路径

在垂直方向上，热量从芯片层向上传递：

```
芯片层 → 界面层 → 扩散器层 → 散热器层 → 环境空气
```

热量传递过程中，每一层都产生一定的温降。各层的温升取决于该层的热阻和通过的热流量。

## 5. 数学公式推导

### 5.1 稳态方程

在稳态条件下（温度不随时间变化），能量守恒方程简化为：

```
∇ · (k · ∇T) + Q = 0
```

对于离散化的块模型，每个块 i 的稳态热平衡方程为：

```
P_i = Σⱼ (T_i - T_j) / R_ij
```

其中：
- P_i 是块 i 的功耗（W）
- T_i 是块 i 的温度（K）
- R_ij 是块 i 和块 j 之间的热阻（K/W）

写成矩阵形式：

```
B · T = P
```

其中：
- **B** 是热导矩阵（nxn 矩阵）
- **T** 是温度向量（nx1）
- **P** 是功率向量（nx1）

矩阵 B 的构建规则：
- 对角元素 B[i][i]：节点 i 的所有热导之和
- 非对角元素 B[i][j]（i ≠ j）：节点 i 和 j 之间的互热阻的负值

```
B[i][i] = Σⱼ 1 / R_ij
B[i][j] = -1 / R_ij
```

### 5.2 瞬态方程

在瞬态条件下，需要考虑热容效应。离散化的瞬态能量守恒方程为：

```
C_i · dTᵢ/dt = P_i - Σⱼ (T_i - T_j) / R_ij
```

其中 C_i 是节点 i 的等效热容（J/K）。

写成矩阵形式：

```
A · dT/dt + B · T = P
```

其中：
- **A** 是热容对角矩阵（nxn 矩阵）
- **B** 是热导矩阵（与稳态方程中相同）
- **P** 是功率向量（W）

如果 A 是可逆的，可以写成标准状态空间形式：

```
dT/dt = A⁻¹ · P - A⁻¹ · B · T = f(T, t)
```

### 5.3 时间离散化

使用后向欧拉法（Backward Euler）对时间进行离散化：

```
(dT/dt) ≈ (Tⁿ⁺¹ - Tⁿ) / Δt
```

代入瞬态方程：

```
A · (Tⁿ⁺¹ - Tⁿ) / Δt + B · Tⁿ⁺¹ = Pⁿ⁺¹
```

整理得：

```
(A/Δt + B) · Tⁿ⁺¹ = A/Δt · Tⁿ + Pⁿ⁺¹
```

设 C = A/Δt + B，则：

```
C · Tⁿ⁺¹ = A/Δt · Tⁿ + Pⁿ⁺¹
```

这是一个线性方程组，可以利用之前的 LU 分解结果快速求解。

## 6. 热阻计算

### 6.1 垂直方向热阻

热量在垂直方向上穿过各层时，每层的热阻为：

```
R_i = tᵢ / (kᵢ · A)
```

其中：
- tᵢ 是第 i 层的厚度（m）
- kᵢ 是第 i 层的热导率（W/(m·K)）
- A 是截面积（m²）

总垂直热阻为各层热阻之和：

```
R_vertical = Σᵢ Rᵢ = Σᵢ tᵢ / (kᵢ · A)
```

### 6.2 水平方向热阻（块间）

相邻块之间的水平热传递通过并联传导机制计算。块 i 和块 j 之间的等效热导为：

```
G = (1/Gᵢ + 1/Gⱼ)⁻¹
```

其中：

```
Gᵢ = k · L_shared / (t · dᵢ)
Gⱼ = k · L_shared / (t · dⱼ)
```

- L_shared 是两块的共享边界长度（m）
- t 是层厚度（m）
- dᵢ、dⱼ 是块 i、j 的等效扩散长度（m）
- k 是材料热导率（W/(m·K)）

水平热阻为：

```
R_lateral = 1 / G
```

### 6.3 接触热阻

层与层之间的界面存在接触热阻，通常由表面粗糙度和界面材料决定。HotSpot 中通过经验公式或测量值来设定。

## 7. 热容计算

### 7.1 单层热容

第 i 层材料的热容为：

```
Cᵢ = ρᵢ · cₚᵢ · Vᵢ = ρᵢ · cₚᵢ · (tᵢ · Aᵢ)
```

其中：
- ρᵢ 是密度（kg/m³）
- cₚᵢ 是比热容（J/(kg·K)）
- Vᵢ 是体积（m³）
- Aᵢ 是面积（m²）

### 7.2 常用材料热容

| 材料 | 密度 ρ (kg/m³) | 比热容 cₚ (J/(kg·K)) | 热导率 k (W/(m·K)) |
|------|----------------|---------------------|-------------------|
| 硅 | 2330 | 700 | 150 |
| 铜 | 8960 | 386 | 400 |
| TIM | 2000 | 1000 | 5 |
| 铝 | 2700 | 900 | 237 |

### 7.3 节点等效热容

对于网格模型，每个节点代表一个有限体积，其等效热容为该体积内材料的热容之和。

## 8. 数值方法

### 8.1 线性方程组求解

稳态方程 B · T = P 和瞬态离散化后的方程 C · Tⁿ⁺¹ = RHS 都涉及求解线性方程组。

HotSpot 采用 LU 分解法：

1. 对矩阵 B 或 C 进行 LU 分解：B = L · U
2. 求解 L · y = P（前向代入）
3. 求解 U · T = y（后向代入）

对于瞬态问题，如果时间步长 Δt 保持不变，可以复用 LU 分解结果，只需更新右侧向量 RHS。

### 8.2 初始条件

瞬态仿真需要设定初始温度分布。HotSpot 中：
- 默认初始温度为环境温度 Tₐₘᵦ
- 也可以从文件读取初始温度分布

### 8.3 时间步长选择

时间步长 Δt 的选择影响数值稳定性和精度：

- Δt 过小：计算精度高，但仿真时间长
- Δt 过大：可能导致数值不稳定

HotSpot 中的时间步长控制：
- 根据热容和热阻估计特征时间：τ = R · C
- 通常取 Δt < min(τᵢ) / 10 以保证精度
- 提供用户可配置的时间步长选项

### 8.4 收敛判定

对于瞬态仿真，通常采用固定时间步长进行迭代计算。对于稳态仿真，判定收敛的条件为：

```
|Tⁿ⁺¹ - Tⁿ| < ε
```

其中 ε 是设定的容差（通常为 0.01 K）。

## 9. 与源代码的对应关系

### 9.1 热阻计算

热阻计算由 `getr()` 函数完成，位于 [src/RCutil.c](../../src/RCutil.c)。

```c
double getr(int m, int n, flp_t *flp, R_model_t *rcd)
```

参数说明：
- m, n：两个节点的索引
- flp： floorplan 数据结构指针
- rcd：热阻模型数据结构指针

### 9.2 热容计算

热容计算由 `getcap()` 函数完成，位于 [src/RCutil.c](../../src/RCutil.c)。

```c
double getcap(int n, flp_t *flp, RC_model_t *rcm)
```

参数说明：
- n：节点索引
- flp： floorplan 数据结构指针
- rcm：RC 模型数据结构指针

### 9.3 矩阵构建

块模型的热阻和热容矩阵构建由 `populate_R_model_block()` 函数完成，位于 [src/temperature_block.c](../../src/temperature_block.c)。

```c
void populate_R_model_block(RC_model_t *rcm, flp_t *flp)
```

该函数：
1. 分配并初始化热阻矩阵 R 和热容矩阵 C
2. 计算所有块对之间的热阻
3. 计算每个块的热容
4. 构建热导矩阵 B

### 9.4 稳态求解

稳态温度求解由 `steady_state_temp_block()` 函数完成，位于 [src/temperature_block.c](../../src/temperature_block.c)。

```c
int steady_state_temp_block(RC_model_t *rcm, flp_t *flp, 
                            double *power, double *temp)
```

参数说明：
- rcm：RC 模型指针
- flp： floorplan 指针
- power：功率向量（W）
- temp：返回的温度向量（K）

返回值：成功返回 0，失败返回错误码

### 9.5 瞬态求解

瞬态温度求解由 `compute_temp_block()` 函数完成，位于 [src/temperature_block.c](../../src/temperature_block.c)。

```c
int compute_temp_block(RC_model_t *rcm, flp_t *flp, 
                      double *power, double *temp, 
                      double t, double dt)
```

参数说明：
- rcm：RC 模型指针
- flp： floorplan 指针
- power：功率向量（W）
- temp：输入输出温度向量（K）
- t：当前仿真时间（s）
- dt：时间步长（s）

该函数使用后向欧拉法进行时间离散化，并调用线性方程组求解器。
